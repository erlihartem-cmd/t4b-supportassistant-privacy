T4B.SupportAssistant — Privacy & Security

This Chrome extension operates locally and does not collect, track, or transmit any personal data. All functionality is limited to user-initiated actions within the browser and local system.

Overview

The extension does not collect or store any personal information, browsing history, or analytics data. All actions occur within your local environment, with optional communication only to:

  • http://127.0.0.1/* — local SupportAssistant backend
  • https://tools4brokers.passwork.io/* — secure Passwork integration

Data Access & Permissions

  • storage — local settings storage only
  • tabs / activeTab — tab control for user-triggered actions
  • scripting — limited to Passwork pages for automation
  • contextMenus — adds contextual actions (e.g., Connect via RDP)
  • alarms — schedules local polling
  • clipboardRead / clipboardWrite — temporary credential copy-paste
  • host_permissions — restricted to Passwork and localhost

Security

  • No third-party analytics or ads
  • No remote code execution
  • All scripts bundled within the extension
  • Local-only processing and automation
